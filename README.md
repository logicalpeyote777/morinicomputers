# Morini Computers — Scripts de los tutoriales

Scripts educativos usados en los tutoriales de **Morini Computers** (YouTube): Proxmox / homelab y
**Ciberseguridad** (hacking ético en laboratorio aislado y propio).

> ⚠️ **Solo con fines educativos.** Todo el contenido de ciberseguridad se ejecuta en un laboratorio
> **AISLADO y de tu propiedad** (red sin salida, máquinas víctima propias). Nunca contra sistemas de terceros.

## Estructura

- `ciberseguridad/<id>-<mod>/` — scripts de cada tutorial de ciberseguridad (p.ej. `ciberseguridad/cyber03-recon/`).
- `proxmox/<id>-<mod>/` — scripts de cada tutorial de Proxmox/homelab.

Cada carpeta corresponde a **un vídeo** y se enlaza desde su descripción en YouTube. Antes de ejecutar
cualquier script en el vídeo se muestra y se explica: el script es contenido, no una caja negra.
